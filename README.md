API Guard is a dev tool developed with the intention to be integrated into CI tests, development just concluded at API level, development is kicking off for the frontend at a rapid pace to bring API Guard to developers as soon as can be.

FEATURES

1. API Guard allows devs to easily and quickly test their API endpoints against vulnerablities that affects a huge chunk of quick ship quick slop API's and projects churned out these days. API Guard supports 6 independent API tests with more advanced and complex tests scheduled to be added after launch.
   Here's some of the tests that will be supported at launch
   _ Idempotency
   _ Race conditions
   _ Rate Limits
   _ Mass assignments
   _ SQL Injections
   _ Broken Object Level Authorization

We're aware that API Gard is an amazingly powerful tool in the palms of bad actors hence we've adopted a series of tough and robust security measures and continue to iterate and improve safety to avoid exploitation.
